# 🌕 전란 그리고 요괴

조선시대 양란(임진왜란, 병자호란) 직후의 혼란한 시기를 배경으로,  
요괴와 원혼을 처치하며 마을의 불미스러운 일을 해결해 나가는 **텍스트 기반 턴제 RPG 게임**입니다.

---

## 🎨 **게임의 특징**

- 🇰🇷 **한국적 배경**: 조선시대 양란 이후 혼란기
- 👹 **전통 요괴**: 한국 전설의 요괴들과 전투
- 🗺️ **지역 탐험**: 조선 팔도를 모티브로 한 8개 지역
- ⚔️ **전략적 전투**: 스킬, 아이템, 상태이상의 조합
- 🔢 **다중 전투**: 1-3마리 몬스터와의 전략적 다중 전투 🆕
- 📈 **성장의 재미**: 레벨업으로 점점 강해지는 캐릭터
- 🎭 **직업별 개성**: 각기 다른 플레이 스타일
- 🎲 **랜덤 요소**: 지역별 랜덤 요괴 출현과 다중 스폰 시스템

---
## 🎮 **게임 실행 방법**

```bash
python main.py
```

## ✨ **완성된 주요 기능들**

### 💾 **세이브/로드 시스템** 🆕
- **3개 슬롯**: 최대 3개의 다른 세이브 파일 관리
- **슬롯 정보 표시**: 플레이어 이름, 레벨, 지역, 저장 시간 표시
- **덮어쓰기 확인**: 기존 데이터 보호를 위한 확인 과정
- **세이브 삭제**: 불필요한 세이브 파일 삭제 기능
- **JSON 기반**: UTF-8 인코딩으로 한글 완벽 지원

```
═══════════════════════════════════════════
                💾 세이브 슬롯
═══════════════════════════════════════════
슬롯 1: 홍길동 (Lv.5) - 한양
        저장 시간: 2024-01-15 14:30

슬롯 3: 임꺽정 (Lv.3) - 제물포

═══════════════════════════════════════════
```

### 🎯 **퀘스트 시스템** 🆕
5개의 메인 퀘스트와 3가지 퀘스트 타입으로 구성된 완전한 퀘스트 시스템:

#### 퀘스트 목록
1. **🌟 한양의 불빛** - 도깨비불 3마리 처치 (경험치 300, 대형 약초)
2. **🌿 의심스러운 약초상** - 소형 약초 5개 수집 (경험치 200, 마력 물약)
3. **🌊 붉은 물속의 그림자** - 검은물귀 처치 (경험치 500, 마력 물약 2개)
4. **💀 무덤의 수호자** - 묘혈수호령 처치 (경험치 600, 대형 약초, 귀한 뼛조각)
5. **🚶‍♂️ 안개를 지나** - 소머리골 도달 (경험치 400, 마력 물약)

#### 퀘스트 특징
- **3가지 타입**: 처치형, 수집형, 이동형
- **조건부 해금**: 레벨 제한, 선행 퀘스트 완료 필요
- **실시간 진행률**: 시각적 진행률 바와 상세 정보 표시
- **자동 추적**: 몬스터 처치, 아이템 수집, 지역 이동 시 자동 진행
- **지역별 제한**: 해당 지역에서만 수락 가능

```
⚔️ 처치: 도깨비불 [■■■■■■■■■■□□□□□] 2/3 (67%)
```

### 📈 **고급 경험치 시스템** 🆕
정교한 3단계 성장률을 적용한 레벨링 시스템 (최대 150레벨):

#### 성장 구조
- **1~49레벨**: 빠른 지수 증가 (초보자 친화적)
- **50~99레벨**: 완만한 지수 증가 (중급자 단계)
- **100~150레벨**: 로그 증가 (엔드게임 컨텐츠)

#### 시스템 특징
- **스마트 스탯 증가**: 레벨 구간별 차등 보너스 (초반 강화, 후반 완만)
- **시각적 진행률**: 실시간 경험치 바와 다음 레벨까지 필요 경험치 표시
- **추천 시스템**: 현재 레벨에 맞는 최적 사냥터 및 몬스터 추천
- **성능 최적화**: 레벨 계산 캐싱으로 빠른 처리

```
레벨 7 [■■■■■■■■■■■■□□□□□□□□] 63.3%
다음 레벨까지: 79 EXP
```

### 🎭 **로딩 화면 시스템** 🆕
조선시대 분위기를 살린 몰입감 있는 로딩 화면:

#### 로딩 타입
- **게임 시작**: 고서 먼지 털기, 등불 밝히기, 성문 열기 등
- **지역 이동**: 지역별 특색 있는 메시지 (바닷바람, 산골 기운 등)
- **전투 시작**: 무기 준비, 기운 모으기 등
- **세이브/로드**: 데이터 처리 과정 시각화

#### 특징
- **지역별 맞춤 메시지**: 한양(궁궐), 제물포(바다), 소머리골(음산함) 등
- **랜덤 명언**: 조선시대 속담과 한자 성어 표시
- **시각적 진행률**: 로딩 바와 퍼센티지 표시

### 👹 **확장된 몬스터 시스템** 🆕
기존 10종에서 25종으로 대폭 확장된 몬스터 시스템:

#### 잡몹 (15종) - 일반 전투용
- **깨비, 좀도깨비**: 장난스러운 도깨비 계열
- **물귀신, 산령**: 자연 정령 계열
- **여우요괴, 까마귀요괴**: 동물 요괴 계열
- **장화홍련, 처녀귀신**: 원혼 계열
- **솥귀신, 바람개비귀신**: 일상 용품 정령 계열
- **출현률**: 20~75% (자주 만남)
- **경험치**: 15~50 (빠른 성장)

#### 중급 몬스터 (10종) - 도전적 전투용
- **어둑시니, 불목도리**: 고유 능력을 가진 강력한 요괴
- **검은물귀, 도끼잡귀**: 지역 특화 보스급 몬스터
- **묘혈수호령**: 최강급 수호 영령
- **출현률**: 5~20% (희귀 조우)
- **경험치**: 100~200 (큰 성장)

### ⚔️ **다중 몬스터 전투 시스템** 🆕
- **확률적 다중 스폰**: 전투마다 최소 1마리, 최대 3마리 몬스터 출현
- **지역별 차등 확률**: 위험도에 따라 다중 스폰 확률 상이
  - 🟢 **안전 지역**: 85% (1마리), 15% (2마리)
  - 🟡 **높은 위험**: 70% (1마리), 30% (2마리)
  - 🔴 **매우높음**: 50% (1마리), 40% (2마리), 10% (3마리)
  - ⚫ **극도로높음**: 30% (1마리), 50% (2마리), 20% (3마리)
- **무리 사냥 특성**: 들개령 등 특정 몬스터의 집단 출현 확률 증가
- **스마트 구별 시스템**: 같은 종류 몬스터 여러 마리 시 자동 번호 부여 (예: 들개령 #1, #2)
- **다중 타겟 전투**: 공격/스킬 대상 선택, 개별 몬스터 상태 추적
- **누적 경험치**: 모든 처치 몬스터 경험치 합산 (최대 3배 + 지역 보너스)

### 🔮 **고급 전투 시스템**
- **턴제 전투**: 플레이어와 적이 번갈아가며 행동
- **5가지 행동**: 공격, 스킬 사용, 아이템 사용, 대상 선택, 도망
- **동적 피해 계산**: `공격력 - 방어력` 기반 (최소 1 피해)
- **전략적 도망**: 가장 빠른 적 기준 속도 비교로 성공률 결정
- **실시간 상태 표시**: 모든 살아있는 적의 HP 및 상태이상 표시

#### 도사 스킬

- **서리파동**: 빙결 상태이상 유발 확률 (MP 4)

#### 유랑객 스킬
- **급소 찔러내기**: 높은 치명타 피해 (MP 6)
- **독 단검**: 독 상태이상 유발 확률 (MP 4)

### 🌟 **상태이상 시스템**
상태이상은 3턴 지속되며 각각 고유한 효과를 가집니다:

- **🔥 화상**: 매 턴 최대 HP의 10% 피해
- **☠️ 독**: 매 턴 최대 HP의 8% 피해  
- **🧊 빙결**: 행동 불가 상태 (실제 행동 차단)
- **😵 기절**: 행동 불가 상태 (실제 행동 차단)

### 📈 **레벨업 시스템**
기존 단순 시스템에서 3단계 성장률을 적용한 고급 시스템으로 진화:

#### 경험치 계산 공식 🆕
- **1~49레벨**: `100 × (1.18^레벨)` - 빠른 초기 성장
- **50~99레벨**: `복합 지수 함수` - 완만한 중기 성장
- **100~150레벨**: `로그 함수` - 극도로 어려운 엔드게임

#### 레벨업 보상
- **HP**: +10 (즉시 회복)
- **공격력**: +2 (레벨 구간별 보너스 적용)
- **방어력**: +1 (레벨 구간별 보너스 적용)
- **최대 MP**: +5
- **스탯 보너스**: 높은 레벨일수록 추가 보너스 배율 적용

### 🔋 **MP 회복 시스템**
- **자동 회복**: 매 턴 종료 시 MP 2씩 회복
- **최대 MP**: 30 + (레벨-1) × 5
- **MP 표시**: 전투 UI에서 현재/최대 MP 표시

### 🎒 **아이템 & 인벤토리 시스템**

#### 기본 회복 아이템
- **소형 약초**: 체력 25 회복
- **대형 약초**: 체력 50 회복
- **마력 물약**: 마력 15 회복

#### 시작 지급 아이템
- 소형 약초 × 3
- 마력 물약 × 2

#### 아이템 사용
- 전투 중 "3. 아이템 사용" 선택
- 인벤토리에서 아이템 선택 후 즉시 효과 적용
- 체력/마력이 가득 찬 상태에서는 사용 제한

### 🗺️ **지역 탐험 시스템**
조선 팔도를 모티브로 한 8개 지역을 자유롭게 탐험할 수 있습니다:

#### 주요 지역
- **🏰 한양**: 조선의 중심지, 모든 상점 이용 가능, 안전한 거점
- **⚓ 제물포**: 서해안 항구도시, 바다를 통한 탐라국 연결
- **⚔️ 한밭**: 요괴가 자주 출몰하는 위험지역, 경험치 +20% 보너스
- **🛤️ 아우내**: 교통의 요충지, 이동비용 할인
- **🌿 빛고을**: 문화 중심지, 약초 할인, 의료 서비스
- **🌋 가마뫼**: 화산지형, 화속성 요괴 출몰, 온천 (MP 회복 +50%)
- **💀 소머리골**: 극도로 위험한 산골, 정신력 -10 페널티, 경험치 +50%
- **🏝️ 탐라국**: 신비로운 섬나라, 경험치 +100%, 배로만 접근 가능

#### 지역 특징
- **연결성**: 인접한 지역으로만 이동 가능
- **위험도**: 각 지역마다 다른 위험 수준과 보상
- **특수 기능**: 상점, 항구, 온천 등 지역별 고유 시설
- **다중 스폰 확률**: 위험도에 따라 몬스터 무리 출현 확률 상이

### 👹 **완전한 요괴 생태계** 🆕
조선시대 설화를 바탕으로 한 25종의 요괴가 서식하는 완전한 생태계:

#### 🐾 잡몹 계열 (15종) - 일상에서 만나는 요괴들
**도깨비 계열**
- **깨비**: 장난스러운 어린 도깨비 (출현률 75%, 경험치 15)
- **좀도깨비**: 작지만 약빠른 도깨비 (출현률 60%, 경험치 25)

**자연령 계열**
- **물귀신**: 물에 빠진 혼령 (출현률 50%, 경험치 30)
- **산령**: 산속을 지키는 정령 (출현률 45%, 경험치 35)
- **들쥐정령**: 들판의 작은 정령 (출현률 70%, 경험치 20)

**동물 요괴**
- **여우요괴**: 변신술을 쓰는 구미호 새끼 (출현률 30%, 경험치 40)
- **까마귀요괴**: 불길한 소식을 전하는 새 (출현률 40%, 경험치 35)
- **무당개구리**: 비를 부르는 신령한 개구리 (출현률 25%, 경험치 45)

**원혼 계열**
- **떠돌이 혼**: 떠도는 무명 혼령 (출현률 55%, 경험치 25)
- **장화홍련**: 억울하게 죽은 자매의 혼 (출현률 20%, 경험치 50)
- **처녀귀신**: 한을 품고 죽은 처녀 혼령 (출현률 35%, 경험치 40)

**일상 정령**
- **솥귀신**: 부엌 솥에 깃든 정령 (출현률 40%, 경험치 30)
- **삿갓귀신**: 버려진 갓에 깃든 혼령 (출현률 45%, 경험치 35)
- **장승귀신**: 마을 수호신이 변한 존재 (출현률 30%, 경험치 45)
- **바람개비귀신**: 바람에 돌아가는 원혼 (출현률 35%, 경험치 40)

#### 👺 중급 몬스터 계열 (10종) - 강력한 요괴들
**고위 요괴**
- **어둑시니**: 어둠을 다루는 상급 요괴 (출현률 15%, 경험치 150)
- **불목도리**: 목을 자유자재로 늘리는 요괴 (출현률 10%, 경험치 180)
- **해골장수**: 뼈를 파는 해골 상인 (출현률 12%, 경험치 160)

**지역 수호령**
- **검은물귀**: 깊은 물속의 검은 정령 (출현률 8%, 경험치 200)
- **도끼잡귀**: 도끼를 휘두르는 산속 잡귀 (출현률 10%, 경험치 170)
- **묘혈수호령**: 무덤을 지키는 최강 수호령 (출현률 5%, 경험치 200)

**특수 능력자**
- **광포살이**: 광기에 빠진 살육자 (출현률 12%, 경험치 165)

4. **기본 지급**: 소형 약초 3개, 마력 물약 2개 지급

### 💾 세이브/로드 활용법 🆕
- **자주 저장**: 중요한 순간마다 진행 상황 저장
- **백업 활용**: 3개 슬롯을 활용한 단계별 백업 전략
- **실험적 플레이**: 위험한 도전 전 별도 슬롯에 저장
- **캐릭터 관리**: 다른 직업으로 새로운 플레이 경험

### 🎯 퀘스트 완수 가이드 🆕
#### 효율적 퀘스트 진행법
1. **퀘스트 확인**: 지역별로 수락 가능한 퀘스트 체크
2. **조건 충족**: 레벨과 선행 퀘스트 완료 여부 확인
3. **동시 진행**: 여러 퀘스트를 동시에 진행하여 효율성 극대화
4. **진행률 추적**: 실시간 진행률 바로 완료 시점 파악

#### 퀘스트별 공략법
- **처치형**: 해당 몬스터가 많은 지역에서 집중 사냥
- **수집형**: 아이템 드롭률이 높은 몬스터 우선 처치
- **이동형**: 최단 경로 계획하여 빠른 완료

### 📈 레벨링 전략 🆕
#### 레벨 구간별 전략
- **1~20레벨**: 안전한 지역에서 기초 실력 쌓기
- **21~49레벨**: 다양한 지역 탐험, 퀘스트 병행
- **50~99레벨**: 중급 몬스터 도전, 효율적 사냥터 활용
- **100~150레벨**: 최고 난이도 지역, 보스급 몬스터 집중

#### 경험치 최적화
- **지역 보너스**: 위험 지역에서 +50~100% 경험치 획득
- **퀘스트 보상**: 단순 사냥보다 높은 경험치 효율
- **몬스터 선택**: 중급 몬스터(희귀)와 잡몹(일반) 균형 있는 사냥

### 👹 확장된 몬스터 시스템 활용법 🆕
#### 몬스터 타입별 공략
- **잡몹 (15종)**: 빠른 레벨업과 아이템 파밍용
  - 높은 출현률로 안정적 성장
  - 초보자도 쉽게 처치 가능
- **중급 몬스터 (10종)**: 도전과 큰 보상
  - 희귀 출현으로 큰 경험치 획득
  - 고급 아이템과 특수 재료 드롭

#### 효율적 사냥법
1. **지역별 특화**: 각 지역의 고유 몬스터 파악
2. **출현률 활용**: 잡몹으로 안정적 성장, 중급으로 도약
3. **저항성 고려**: 몬스터별 약점 파악하여 스킬 선택

### 다중 전투 가이드
1. **몬스터 수 확인**: 1-3마리까지 다양하게 출현
2. **대상 선택**: 공격/스킬 사용 시 타겟 선택 가능
3. **우선순위 전략**: 약한 적부터 vs 강한 적 집중 선택
4. **상황 판단**: 불리할 때 전략적 도망 고려
5. **경험치 최대화**: 모든 적 처치 시 누적 보상

### 전투 가이드
1. **기본 공격**: 안정적인 피해, MP 소모 없음
2. **스킬 사용**: 강력하지만 MP 소모, 상태이상 유발 가능
3. **아이템 사용**: 체력/마력 회복, 전략적 타이밍 중요
4. **도망하기**: 속도에 따라 성공률 결정, 유랑객이 가장 유리

### 탐험 전략
- **안전 지역**: 한양에서 회복 및 보급
- **경험치 농장**: 한밭, 소머리골에서 고효율 레벨업
- **다중 전투 활용**: 위험 지역에서 3마리 동시 처치로 최대 경험치
- **특수 지역**: 가마뫼 온천에서 MP 회복, 탐라국에서 최고 경험치
- **지역 연결**: 지도를 숙지하여 효율적인 이동 경로 계획
